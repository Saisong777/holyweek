# 聖週體驗 Holy Week Experience

**iM行動教會 WeChurch**

從棕枝主日到復活節，走過耶穌生命最後一週的沉浸式互動旅程。

## Features

- 🌿 8 天完整聖週旅程（棕枝主日 → 復活節）
- 🎭 每日選擇劇場 — 置身聖經場景做出選擇
- 📖 和合本經文 + 四聲道深度查經
- 🔍 護教議題 — 針對不同受眾的歷史/神學探討
- 💬 匿名故事牆 — 社群共鳴與回饋
- 📬 給週六自己的信 — 密封至復活節揭曉
- 🍷 最後晚餐共食指南
- 🔥 小組帶領工具（計時器、討論題、帶領提示）
- 📱 分享卡片產生器（1080×1080）
- ✝️ 受難日黑屏模式（下午 3-6 點）

## 三種體驗路徑

| 路徑 | 對象 | 特色 |
|------|------|------|
| 🌍 好奇者 | 非信徒 / 初次接觸 | 故事化敘事、零術語 |
| 📖 追隨者 | 基督徒 | 經文連結、深度查經 |
| 🔥 帶領者 | 小組長 / 傳道人 | 完整查經、討論引導 |
